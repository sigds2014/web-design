    CSS Colors for the starving artist is a reference of the color names that can be used in CSS without a #hex code or rgb(value). It is a guide for designers who are just starting out in HTML or CSS, like me (what have we been doing all this time?!). The collection deals directy with the CSS colors named after foods based on age old stereotype of the starving artist, but I'd like to think that those artists could afford a nice juicy steak dinner if they picked up some HTML/CSS skills to boast.

    Our collection of color options is divided into the five major food groups. Hovering over the colored circles reveals color palettes made using only CSS Colors.

    See a list of all of the CSS Colors at the w3schools site here.
